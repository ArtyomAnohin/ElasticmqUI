# Web client for manage ElasticMQ

###Getting started

Install docker and run:
```
docker-compose up
docker-compose stop
```
Otherwise, for the standalone web service:
```
pip install -r requirements.txt
python start.py
```
Visit [http://localhost:5000](http://localhost:5000)
